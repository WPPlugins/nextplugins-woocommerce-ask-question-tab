== Description ==

Ask question tab with mail form for WooCommerce product page.

**Compatible with WooCommerce 2.6 and 3.0**

Plugin include three templates:
Default - for WordPress official themes
Bootstrap - for themes built with bootstrap
Storefront - for Storefront theme

Installation:

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to WooCommerce -> Settings -> Ask Question Tab -> Enable -> Save
